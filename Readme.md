# Invariant Coordinate Selection: An Overview and Recent Developments

This project contains code to reproduce the examples of the paper:

Archimbaud, A., Nordhausen, K. and Ruiz, A.M. (2026): Invariant Coordinate Selection: An Overview and Recent Developments. Submitted


# Reproduce results

The main code files, written in R, are in the folder **scripts**:

- *run_illustrative_examples.R*: R code to reproduce Figures 1 and 3.
- *run_clustering_crabs.R*: R code to reproduce Figure 4.
- *run_outlier_detection_HTP.R*: R code to reproduce Figure 5.

In addition in the folder **figures** all ready figures are provided as pdfs.
